# Figma UI 감사

기준 파일은 `카카오 테크 부트캠프`의 `커뮤니티_GUI_완성` 페이지이며, 기준 뷰포트는 1920 × 1080이다. Figma 메타데이터, 디자인 컨텍스트, 스크린샷을 함께 확인했다.

## 카드와 미디어

- 피드 카드 외곽 폭 480px, 내부 448px, 패딩 16px, 카드 상단 경계선 1px.
- 가로 이미지는 448 × 288, `aspect-ratio: 280 / 180`, 반경 12px.
- 세로 이미지는 448 × 600.85 수준이며 원본 비율을 유지한다.
- 작성자 avatar는 34px, 내부 아이콘은 24px, 테두리는 1.5px.
- 좋아요·댓글·북마크 아이콘은 20px이고 액션 터치 영역은 36px 수준이다.
- 상세 댓글 작성 영역과 각 댓글 카드는 콘텐츠 양쪽에서 16px 떨어진 448px 실사용 폭을 사용한다.
- 생성·편집 모달의 56px 하단 액션 영역은 448 × 44px이며 카메라와 완료 버튼은 각각 32px다. 카메라는 내부 왼쪽 16px, 완료 버튼은 내부 오른쪽 6px에 놓인다.
- 생성·편집 모달 헤더는 52px이고 본문 410px, 하단 액션 56px을 합쳐 480 × 518px이다.
- 프로필·비밀번호 제목은 Title3/Bold(20px/28px)를 사용한다. 사용자 요청에 따라 모달 중앙 축에 맞췄다.

## 타이포그래피와 색상

- 폰트: Wanted Sans.
- Title3/Bold: 20px / 28px / 700 / -0.12px.
- Body1/Bold: 16px / 24px / 700 / 0.057px.
- Body3: 14px / 20px / 400 또는 500 / 0.145px.
- Label1: 13px / 18px / 400·500·600 / 0.252px.
- Label2: 12px / 16px 또는 18px / 400·600 / 0.252px.
- Primary text/icon `#171717`, secondary `#737373`, tertiary `#a1a1a1`.
- Primary border `#d4d4d4`, secondary border `#e5e5e5`.
- Brand primary `#5b5ceb`, surface `#ffffff`, tertiary background `#f5f5f5`.

위 값은 읽기 전용 `styles/tokens/colors.css`와 `styles/tokens/typography.css`에 이미 존재한다. 컴포넌트 전용 크기와 배치만 수정 가능한 CSS에 선언한다.

## 기준 스크린샷

Figma 원본 13개는 `tests/ui/baselines/figma/`에 로컬 PNG로 보관했다. 임시 Figma 에셋 URL은 앱 코드와 테스트에 사용하지 않는다.
